## צפייה

האתר מוכן ל-GitHub Pages.

1. צור ריפו חדש בשם `austria-2026` (אם עדיין לא קיים)
2. העלה את הקבצים (או דחוף את התיקייה)
3. Settings → Pages → Source: Deploy from a branch → main / (root)
